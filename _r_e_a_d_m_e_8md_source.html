1 # 3 speed Rotary Encoder Half Step library for Arduino
2 [![Build Status](https://travis-ci.org/Erriez/ErriezRotaryEncoderHalfStep.svg?branch=master)](https://travis-ci.org/Erriez/ErriezRotaryEncoderHalfStep)
3 
4 This is an optimized three speed Rotary Encoder library for Arduino which supports:
5 
6 * Half step Rotary Encoder types.
7 * Detect three rotation speeds.
8 * Configurable rotation speed sensitivity.
9 * Polling and interrupt based.
10 * Single or multiple Rotary Encoders.
11 * Optional Rotary button.

17 ## Half step / half step Rotary Encoders
18 
19 The difference between a half step or half step Rotary Encoder type is how the data signals of the two pins are generated. It depends on the mechanical construction of the notches and contacts inside the Rotary Encoder.
20 
21 Please refer to the [ErriezRotaryEncoderFullStep](https://github.com/Erriez/ErriezRotaryEncoderFullStep) library for full step Rotary Encoders.
22 Experiment with the full step and half step libraries which works optimal for your Rotary Encoder.

25 ## Hardware
26 
27 Connect the two rotary pins to the DIGITAL pins of an Arduino board.
28 
29 A third rotary button pin is not used in the Rotary library, but can be used in the sketch.
30 
31 Tested with Arduino IDE v1.8.5 on hardware:
32 
33 * Arduino UNO
34 * Arduino Nano
35 * Arduino Micro
36 * Arduino Pro or Pro Mini
37 * Arduino Mega or Mega2560
38 * Arduino Leonardo
39 * WeMos D1 R2 & mini (ESP8266)

42 ### Interrupts
43 
44 Both rotary pins must be connected to a DIGITAL pin with interrupt support, such as `INT0` or `INT1`. This is chip specific. Please refer to the documentation of your board or [attachInterrupt()](https://www.arduino.cc/reference/en/language/functions/external-interrupts/attachinterrupt/).

61 ### Arduino WeMos D1 R2 & mini (ESP8266) hardware
62 
63 Note that some ESP8266 pins mixes ESP8622 GPIO pins with Arduino digital pins. Connect a Rotary Encoder to the following pins which can be used with polled and interrupt examples:
64 
65 | Rotary pin | ESP8622 pin | Text on board / WeMos D1 & R2 |
66 | :---------------: | :---------: | :---------------------------: |
67 | 1 | GPIO13 | D7 (MOSI) |
68 | 2 | GPIO12 | D6 (MISO) |
69 | Button (optional) | GPIO14 | D5 (SCK) |
70 | LED (Not used) | GPIO2 | D4 |
71 | GND | GND | GND |
72 
73 **Note:** An external pull-up resistor is required when a pin does not have an internal pull-up.
74 
75 ```c++
76 // Connect the rotary pins to the WeMos D1 R2 board:
77 #define ROTARY_PIN1 12
78 #define ROTARY_PIN2 13
79 #define ROTARY_BUTTON_PIN 14
80 ```

101 ## Usage
102 
103 **Read rotary with polling**
104 ```c++
105 #include <ErriezRotaryHalfStep.h>
106 
107 
108 // Connect rotary pins to the DIGITAL pins of the Arduino board
109 #define ROTARY_PIN1 2
110 #define ROTARY_PIN2 3
111 
112 // Enable ONE of the three constructors below with different number of arguments:
113 
114 // Initialize half step rotary encoder, default pull-up enabled, default
115 // sensitive=100
116 RotaryHalfStep rotary(ROTARY_PIN1, ROTARY_PIN2);
117 
118 // Or initialize half step rotary encoder, pull-up disabled, default sensitive=100
119 // RotaryHalfStep rotary(ROTARY_PIN1, ROTARY_PIN2, false);
120 
121 // Or initialize half step rotary encoder, pull-up enabled, sensitive 1..255
122 // A higher value is more sensitive
123 // RotaryHalfStep rotary(ROTARY_PIN1, ROTARY_PIN2, true, 150);
124 
125 void loop()
126 {
127  int rotaryState = rotary.read();
128 
129  // rotaryState = -3: Counter clockwise turn, multiple notches fast
130  // rotaryState = -2: Counter clockwise turn, multiple notches
131  // rotaryState = -1: Counter clockwise turn, single notch
132  // rotaryState = 0: No change
133  // rotaryState = 1: Clockwise turn, single notch
134  // rotaryState = 2: Clockwise turn, multiple notches
135  // rotaryState = 3: Clockwise turn, multiple notches fast
136 }
137 ```
138 
139 **Read rotary with interrupts**
140 
141 ```c++
142 #include <ErriezRotaryHalfStep.h>
143 
144 // Connect rotary pins to Arduino DIGITAL pins with interrupt support:
145 //
146 // +-----------------------------------+--------------------------+
147 // | Board | DIGITAL interrupt pins |
148 // +-----------------------------------+--------------------------+
149 // | Uno, Nano, Mini, other 328-based | 2, 3 |
150 // | Mega, Mega2560, MegaADK | 2, 3, 18, 19, 20, 21 |
151 // | Micro, Leonardo, other 32u4-based | 0, 1, 2, 3, 7 |
152 // +-----------------------------------+--------------------------+
153 //
154 #define ROTARY_PIN1 2
155 #define ROTARY_PIN2 3
156 
157 // Enable ONE of the three constructors below with different number of arguments:
158 
159 // Initialize half step rotary encoder, default pull-up enabled, default
160 // sensitive=100
161 RotaryHalfStep rotary(ROTARY_PIN1, ROTARY_PIN2);
162 
163 // Or initialize half step rotary encoder, pull-up disabled, default sensitive=100
164 // RotaryHalfStep rotary(ROTARY_PIN1, ROTARY_PIN2, false);
165 
166 // Or initialize half step rotary encoder, pull-up enabled, sensitive 1..255
167 // A higher value is more sensitive
168 // RotaryHalfStep rotary(ROTARY_PIN1, ROTARY_PIN2, true, 150);
169 
170 void setup()
171 {
172  // Initialize pin change interrupt on both rotary encoder pins
173  attachInterrupt(digitalPinToInterrupt(ROTARY_PIN1), rotaryInterrupt, CHANGE);
174  attachInterrupt(digitalPinToInterrupt(ROTARY_PIN2), rotaryInterrupt, CHANGE);
175 }
176 
177 void rotaryInterrupt()
178 {
179  int rotaryState = rotary.read();
180 
181  // rotaryState = -3: Counter clockwise turn, multiple notches fast
182  // rotaryState = -2: Counter clockwise turn, multiple notches
183  // rotaryState = -1: Counter clockwise turn, single notch
184  // rotaryState = 0: No change
185  // rotaryState = 1: Clockwise turn, single notch
186  // rotaryState = 2: Clockwise turn, multiple notches
187  // rotaryState = 3: Clockwise turn, multiple notches fast
188 }
189 ```
